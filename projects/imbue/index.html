Imbue

A VR Archery game with engaging, dynamic enemy AI and a elemental reaction system.

Project Details

Solo Project

Duration: 4 Months

Gameplay
Overview
Equipped with a bow and arrows, fight through enemies with the help of magical elements that you can imbue into your arrow.

This game features .
Gallery
Features
  • Multi-part character customization from appearance to quiver position.
  • Control system designed around making combat as intuitive as possible.
  • Element system with distinct effects with reactive interactions between each element.
  • Dynamic and engaging feeling enemies that retreat, strafe, and attack.
  • Satisfying visual and audio effects.
Technical Highlight

Elemental Reactions

Elements have unique effects, like ice that freezes or fire that burns. On top of that, when another element is applied, it should react accordingly to the current effect the enemy has, like Fire + Ice = Wet.
